# 🧭 КОМПАС ПРОЕКТА «seedance-generator»
> Добавлено 2026-06-30, чтобы ЛЮБАЯ сессия Claude сразу знала, где что. Не ищи исходники по Downloads/Desktop — бери из первоисточника ниже.

**Что это:** Генератор видео Seedance (веб-сервер).
**Тип:** Node/JS

## 📍 Где исходники (единый первоисточник)
- **GitHub (главное, приватный):** https://github.com/damonbogdan/seedance-generator
- **На этом диске (Linux, внешний SSD T7):** `/mnt/t7/work/seedance-generator`
- **Склонировать на любой машине (Mac и т.д.):** `gh repo clone damonbogdan/seedance-generator`
- ⚠️ Если на Mac лежит копия в `~/Downloads` или `~/Desktop` — это та же программа. Первоисточник — GitHub-репозиторий выше.

## ▶️ Как запустить
`npm install`, затем `npm start` (=`node server.mjs`). Ключ в .env. Настройки (не секрет) — config.json.

- Ключи/пароли — в `.env` (и подобных), они НЕ в git и НЕ на GitHub.
